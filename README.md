# 🔍 DevLens — AI-Powered Code Reviewer

A full-stack web application that uses Google Gemini AI to review code and provide instant feedback on bugs, performance, security, and improvements.

## ✨ Features
- AI Code Review powered by Google Gemini API
- Bug Detection — finds logical and runtime errors
- Performance Analysis — suggests optimizations
- Security Check — identifies vulnerabilities
- JWT Authentication — secure login and register
- Review History — all reviews saved in database
- Multi-language support — JavaScript, Python, Java, C++, SQL

## 🛠️ Tech Stack
- Frontend: Vanilla JavaScript, HTML5, CSS3, CodeMirror
- Backend: Node.js, Express.js
- Database: PostgreSQL
- AI: Google Gemini API
- Auth: JWT + bcrypt

## ⚙️ Setup

1. Clone the repo
2. Run `npm install`
3. Create `.env` file with your credentials
4. Setup PostgreSQL database
5. Run `npm run dev`
6. Open `http://localhost:3000`
